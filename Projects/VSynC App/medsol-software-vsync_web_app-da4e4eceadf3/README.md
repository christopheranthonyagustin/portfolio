# Medical Early Warning System (MEWS)

## Overview

The Medical Early Warning System (MEWS) is a regulated medical device platform designed for real-time patient monitoring, physiological data collection, clinical decision support, and early detection of patient deterioration.

The platform integrates hardware devices, embedded firmware, cloud services, mobile applications, and backend systems to support healthcare providers in monitoring patient conditions and facilitating timely clinical intervention.

This project was delivered within a regulated medical device environment following ISO 13485 Quality Management System requirements, IEC 62304 software lifecycle processes, and ISO 14971 risk management practices.
